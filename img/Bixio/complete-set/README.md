# Bixio Complete Set

Questa cartella contiene tutte le foto presenti nello zip originale di Bixio, rinominate con nomi coerenti e ottimizzate per il web.

I file gia in uso dal sito restano in `img/Bixio/`.
Qui dentro trovi il set completo da cui possiamo scegliere e riordinare facilmente le immagini della pagina Bixio.

## Note rapide

- `livingroom`: salotto
- `kitchen`: cucina
- `day-area`: zona giorno con vista aperta
- `dining-area`: zona pranzo
- `bedroom`: camera da letto
- `bathroom`: bagno
- `exterior`: balcone / esterno
- `additional`: foto aggiuntive interne
- `entrance`: ingresso edificio / appartamento
- `street`: vista strada e accesso
